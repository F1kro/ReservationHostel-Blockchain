# Web Reservasi Hotel - dApp (ethers.js)

Projek ini adalah contoh sederhana aplikasi reservasi hotel berbasis blockchain.
Struktur proyek dan instruksi lengkap disertakan. Untuk memulai:

1. Pastikan Node.js & npm terpasang.
2. `npm install` untuk menginstal dependensi.
3. Compile: `npx hardhat compile`.
4. Deploy lokal: jalankan `npx hardhat node` di terminal lain, lalu `npx hardhat run scripts/deploy.js --network localhost`.
5. Salin address kontrak & ABI ke `frontend/app.js`.
6. Buka `frontend/index.html` di browser dengan MetaMask terhubung ke network yang sama.

Jika Anda ingin, saya bisa:
- Menambahkan versi React.
- Menambahkan fitur pembatalan/reschedule.
- Membantu deploy ke testnet.
